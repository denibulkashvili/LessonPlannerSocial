Lesson Planner is an application for faster lesson planning for ESL classes.

Made for teachers by teachers

This project is free and always in develepment so new features will roll out regularly

Current Features

  • check Basic account functionality: signup, login. logout, change password
  • check Create, edit and delete new lesson plans
  • check Filter lesson plans by tags
  • check Filter lesson plans by books
  • check Add YouTube video url and preview a video on lesson plan details page

Future Features

  • check_box_outline_blank WYSIWYG editor in lesson plan creation/editing form
  • check_box_outline_blank Recommendations: On lesson creating form, pick tags to see recommended related activities or games
  • check_box_outline_blank File uploads: upload worksheets, flashcards and other files
  • check_box_outline_blank Templates: save lessons in .doc or .pdf
  • check_box_outline_blank Custom templates: create your own templates with your name/school logo etc.
  • check_box_outline_blank Reports: section for interactive and fast student reports creation
  • check_box_outline_blank Games/Activities separate section: New section dedicated to classroom games/activities only
  • check_box_outline_blank Comments: post comments lesson plans, activites
  • check_box_outline_blank Advanced accounts: add personal info, avatar, list of favorite lesson plans/activities
  • check_box_outline_blank Voting for games and activities: rate activities and help us find the best ones, save to favorites
  • check_box_outline_blank Badges: Receive badges for created lesson plans or adding highly rated activities
  • check_box_outline_blank Social auth: login with your social media account
